Privacy Policy for CNF Price Calculator

Effective date: June 17, 2025

1. Introduction

CNF Price Calculator is a Chrome extension built to facilitate price calculation for people working in a vehicle exporting company. This Privacy Policy outlines how we collect, use, and share information when you use the extension.

2. Data Collection

This extension collects:

  • Username: Provided by you to assign and verify your license.
  • Chrome Unique ID: A unique identifier automatically generated by Chrome to associate your license and usage.

No other personal data (such as browsing history or personal files) is collected, transmitted, or stored.

3. Permissions

In its manifest.json, CNF Price Calculator requests the following permissions:

  • host permission for:
    • Fetching live exchange rates via the Exchange Rate API
    • Retrieving current time via a Time API
    • Authenticating licenses using the Airtable API

4. Third‑Party Services

CNF Price Calculator integrates with the following third‑party services:

  • Exchange Rate API: Retrieves live currency exchange rates for price calculations.
  • Time API: Fetches current time data to ensure accurate rate timestamps.
  • Airtable API: Authenticates and verifies user licenses.

Only the Username and Chrome Unique ID are sent to these services for their respective functions; no additional personal or browsing data is shared.

5. Data Storage & Security

All data is processed in‑browser or transmitted securely over HTTPS to the above APIs. We do not store any user data on our own servers.

6. Children’s Privacy

This extension is not intended for use by children under 13. We do not knowingly collect data from children.

7. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be effective immediately upon posting the revised version, with an updated effective date.
